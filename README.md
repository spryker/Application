# Application Module
[![Build Status](https://travis-ci.org/spryker/application.svg)](https://travis-ci.org/spryker/application)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%207.3-8892BF.svg)](https://php.net/)
